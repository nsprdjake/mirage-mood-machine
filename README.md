# Mirage Mood Machine

A self-contained Palm Springs / desert / casino-inspired vibe generator built as a single `index.html`.

## What it does

Spin three animated reels full of desert icons and it generates a stylish reading with:
- a title
- desert forecast
- lucky number
- soundtrack mood
- suggested mission
- cheeky fortune

It also includes:
- **Surprise Me** for instant random readings
- **Copy Result** to grab the text
- **Download Poster** to save the current reading as a shareable image
- a tiny Bailey easter egg when the dog symbol appears

## How to use it

Open `index.html` in any modern browser and hit **Spin the Mirage**.

No build step, no install, no backend — just one file and a suspicious amount of desert energy.
